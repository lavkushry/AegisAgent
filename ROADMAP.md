# AegisAgent Roadmap

## MVP Launch Readiness

- Local gateway quickstart with Docker Compose.
- Python SDK `@protect_tool` decorator stable enough for demos.
- Default Cedar policy pack for read-only allow, main-merge approval, and untrusted mutation denial.
- GitHub attack demo with audit timeline output.

## Q3 2026

- Python SDK stable release.
- Slack approval callback verification and signed callbacks.
- Dashboard-backed approval queue.
- Expanded GitHub action templates and branch protection examples.
- Audit event redaction and payload capture controls.

## Q4 2026

- TypeScript SDK.
- MCP manifest signing and drift detection.
- MCP proxy execution path, not just authorization controls.
- OpenTelemetry exporter examples.
- GitHub App integration and PR comments/checks.

## Post-MVP

- Policy bundle versioning and dry-run mode.
- Per-tenant rate limiting.
- SIEM/webhook export.
- Helm chart and production deployment hardening.
- Demo video and hosted sandbox environment.
